发表于 2020-03-16 | 分类于 实战

框架选择

用于搭建个人框架的一些选择

  1. wordpress.org 动态博客【前后台】
  2. hexo.io 静态博客
  3. jekylirb.com 静态
  4. gohugo.org 静态 基于go语言
  5. vuepress
  6. solo.b3log.org

实操 – hexo

  1. 下载node.js 安装
  2. npm 比较慢，下载cnpm
  • -g 全局安装
  • registry 镜像源
  1. cnpm install -g hexo-cli
  2. 创建一个博客目录并切换到相应目录
  3. 初始化
  4. 启动，hexo就会给出端口，在浏览器输入地址访问
  5. 创建文章
  6. 编辑相应文章
  7. 清理 hexo clean 生成 hexo g
  8. 重新启动 hexo s
  9. 部署到Github
    • 新建仓库：用户名.github.io
    • 回到命令行安装hexo的一个插件
    • 设置 _config.yml文件
