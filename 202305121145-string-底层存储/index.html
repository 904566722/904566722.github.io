Q&A

变量 a 与 b 在赋值的时候给到了相同的字符串值，底层会指向同一块地址吗（hello的字符串在底层是否只分配一次？）

答案是：是。

func main() {
    a := "hello"
    b := "hello"
    c := b
    e := "world"

    fmt.Println(&a)
    fmt.Println(&b)
    fmt.Println(&c)
    fmt.Println(&e)

    fmt.Println(a==b)
    fmt.Println(b==c)
    fmt.Println("done")
}

两个字符串相加之后如果跟另一个字符串相同，是否指向同一个地址？

答案是：否

查看两个不同的pointer指向的内容，都是”hello”，但是是两块不同的存储空间

这说明了字符串连接的 + 操作是低效的，会一直产生临时的字符串
