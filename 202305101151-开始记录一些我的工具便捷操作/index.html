图床

使用 github 保存图片

编辑的时候粘贴的时候都保存到本地 themes/DoIt/assets/images/posts 文件夹中，写作的时候无需做上传图片的操作

但是需要把 ·themes/DoIt/assets/ images/posts 这段内容删除，为的是让 hugo 生成静态页面之后能找到这张图片

这样就能够同时在 博客、obsidian 中看到这张图片
